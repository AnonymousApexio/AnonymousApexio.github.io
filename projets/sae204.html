SAE2.04 Intégrative

Projet intégrateur en réseau, téléphonie, collecte de données, web et base de données

Voir le code sur GitHub

Description du projet

La SAE Intégrative est un projet multidisciplinaire qui combine quatre domaines essentiels des réseaux et télécommunications :

  • Réseau : Configuration d'un réseau local avec VLAN, routage inter-VLAN, services (DHCP, DNS, FTP, NTP) et sécurité (ACL, iptables)
  • Téléphonie : Mise en place d'une infrastructure VoIP avec téléphones logiciels (Linphone) et matériel, utilisant le protocole SIP
  • Collecte de données : Développement d'un système pour récupérer des données de capteurs via MQTT
  • Web et base de données : Conception d'une interface web avec Django pour visualiser et gérer les données collectées

Ce projet a été réalisé en équipe dans le cadre du BUT Réseaux et Télécommunications, avec pour objectif de valider nos compétences fondamentales dans ces différents domaines.

Méthodologie appliquée

Pour mener à bien ce projet complexe, nous avons adopté une approche méthodique en plusieurs étapes :

  1. Planification de l'infrastructure réseau : Conception des VLAN, allocation des plages IP, définition des politiques de sécurité
  2. Configuration matérielle : Mise en place du routeur (ROAS, DHCP, NAT) et du commutateur (VLAN, port mirroring)
  3. Déploiement des services : Installation et configuration de FTP, NTP, Rsyslog sur le serveur Debian
  4. Partie téléphonie : Configuration des téléphones logiciels (Linphone) et matériel, inscription au service SIP cloud
  5. Développement de la solution de collecte : Création de scripts Python pour récupérer les données MQTT des capteurs
  6. Conception de la base de données : Modélisation des tables pour stocker les informations des capteurs et leurs données
  7. Développement de l'interface web : Création d'un dashboard avec Django pour visualiser et gérer les données
  8. Tests et validation : Vérification complète de toutes les fonctionnalités et interconnectivité des systèmes
